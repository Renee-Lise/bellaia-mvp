# BellaStudio Digital — Vitrine GitHub Pages

## Installation rapide sur GitHub

1. Crée un compte GitHub ou connecte-toi.
2. Clique sur **New repository**.
3. Nom conseillé : `bellastudio-digital`.
4. Mets le dépôt en **Public**.
5. Clique sur **Create repository**.
6. Clique sur **Add file > Upload files**.
7. Dépose tous les fichiers de ce dossier : `index.html`, `styles.css`, `config.js`, `app.js`, etc.
8. Clique sur **Commit changes**.
9. Va dans **Settings > Pages**.
10. Source : **Deploy from a branch**.
11. Branch : **main** / folder : **/root**.
12. Clique sur **Save**.

Ton lien sera du type :
`https://tonnomgithub.github.io/bellastudio-digital/`

## Modifier les liens Ko-fi

Ouvre `config.js` et remplace les `paymentLink` par les liens produits Ko-fi spécifiques quand ils seront créés.

Pour l’instant, tous les boutons renvoient vers :
https://ko-fi.com/viloassistance

## Modifier le numéro WhatsApp
